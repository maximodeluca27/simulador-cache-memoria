## Ejecución del simulador
El archivo principal es `Memoria_Accesos.ipynb`, desarrollado en **Jupyter Notebook / Google Colab**.

Pasos:
1. Abrir el notebook en Jupyter o Google Colab
2. Ejecutar las celdas de configuración
3. Ingresar los parámetros de la caché
4. Simular accesos a memoria
5. Analizar métricas y gráficos generados

---

## Tests
El archivo `tests.py` contiene un conjunto de pruebas diseñadas para validar:
- Funcionamiento correcto de hits y misses
- Reemplazo FIFO
- Comportamiento ante repetición de accesos
- Manejo de capacidad límite de la caché

Estas pruebas permiten verificar la consistencia del simulador ante distintos patrones de acceso.
